# 企业微信小程序易积分系统

生产版项目骨架，包含 Node.js API、Vue 管理后台、uni-app 企业微信小程序端、MySQL 8/Redis/Docker 部署配置。

## 当前实现重点

- 积分流水备注随流水永久留存，后端不提供删除或修改备注接口。
- 单笔奖惩必须填写备注；月度批量录分支持统一备注和单人备注覆盖。
- 员工端积分明细接口按月份返回每条流水的完整备注。
- 申诉必须绑定员工自己的积分流水。
- 订单兑换扣分，驳回/取消可通过冲正流水退回积分。

## 本地验证

当前环境没有 npm 时，也可以直接运行核心后端测试：

```bash
node --test apps/api/tests/*.test.js
```

启动 API：

```bash
node apps/api/src/server.js
```

完整依赖安装后可按各子项目 `package.json` 继续运行管理端和小程序端。
